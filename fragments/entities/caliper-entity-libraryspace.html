LibrarySpace

A Caliper LibrarySpace represents any space used for Library services either permanently or temporarily.

IRI
http://purl.imsglobal.org/caliper/LibrarySpace
Term
LibrarySpace
Supertype
Entity
Properties
LibrarySpace inherits all properties defined by its supertype Entity, of which id and type are required. LibrarySpace is also provisioned with the additional properties BuildingName, BuildingNumber, RoomNumber, floor, servicePoint, isPartOf. Additional type restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to the Term LibrarySpace. Required
RoomNumber string The room number for this space if there is one. Optional
BuildingNumber string The building number for this space if it's in a building. Optional
BuildingName string The building name for this space if it's in a building. Optional
floor string The floor for this LibrarySpace. Generally used to indicate loose areas within a Library. Optional
servicePoint string The name of location where the interaction happened. For example circulation desk, reference desk, wandering ipad" Optional
isPartOf Library | IRI A related Library that this LibrarySpace belongs to. The isPartOf value MUST be expressed either as an object or as a string corresponding to the associated entity's IRI. Optional
