Term
ToolUseProfile

The Caliper Tool Use Profile models an intended interaction between a user and a tool. In other words, when a Person utilizes a SoftwareApplication in a manner that the application determines to be its intended use for learning, an application that implements the Tool Use Profile can emit a ToolUseEvent indicating such usage.

The Tool Use Profile enables the gathering of basic usage information. It provides an easy way to get started with a base level of instrumentation by allowing the learning tool to make the determination of its own use. Any learning app can be instrumented using this profile to detect when a learner accesses the tool and uses it in the way it was intended.

The Tool Use Profile supports the following user stories:

  • As an instructor, I want to know what tools are being used, and how often.
  • As an instructor, I want to know who are the most active and least active users of tools.
  • As an instructor, I want to know whether or not tool usage helps or hinders student performance.
  • As an instructor, I want to know which tools have the greatest impact on student performance.
  • As an instructor, I want to know how much time learners are spending using a tool.
  • As an instructor, I want to know how much curricular progress learners have made in a tool.
  • As an instructor, I want to be able to view my students' "time on task" across multiple learning tools using a dashboard powered by a Caliper event store.
  • As a data scientist, I want to be able to study the impact of a student's curricular progress in a learning tool on his or her academic achievement.
  • As a student, I want to confirm that the progress and time metrics I see in a learning tool matches the metrics I see in an aggregated dashboard.

The Tool Use Profile defines a ToolUseEvent for describing a Person using a learning tool in a way that the tool's creators have determined is an indication of a learning interaction. A ToolUseEvent MAY reference a generated AggregateMeasureCollection containing one or more AggregateMeasure entities that provide information on a learner's "progress" using the tool. The following metrics can be recorded: AssessmentPassed, AssessmentSubmitted, MinutesOnTask, SkillsMastered, StandardsMastered, UnitsCompleted, UnitsPassed, WordsRead.
