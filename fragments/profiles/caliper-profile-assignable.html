AssignableProfile

The Assignable Profile models activities associated with the assignment of digital content to a learner for completion according to specific criteria. Caliper provides a generic AssignableDigitalResource for describing the object of these activities as well as a learner's Attempt for recording a count of the number of times an assigned resource has been attempted.

The Assignable Profile supports the following user stories:

  • As an instructor I want to know who is assigned what material.
  • As an instructor I want to know how long does it take to complete something that has been assigned.
  • As an instructor I want to know what piece of assigned material presents the biggest challenge (i.e. needs most retake attempts).

The Assignable Profile defines an AssignableEvent, NavigationEvent, and ViewEvent for describing learner interactions with assignments.

For each AssignableEvent described, the governing profile SHOULD be referenced (e.g., "AssignableProfile"). Additionally, the generated Attempt SHOULD be specified for the following AssignableEvent actions:

  • Started
  • Completed
  • Submitted
  • Reviewed
Set the count value to 1 for a first attempt and increment by 1 for each subsequent attempt.

If the Attempt is expressed as an object the Attempt SHOULD define, at a minimum, the assignee, the assignable, and the count.

Parent-child relationships that exist between AssignableDigitalResource attempts MAY be represented by use of the Attempt isPartOf property.
