ResourceUseEvent

A Caliper ResourceUseEvent models a Person making use of a library's circulation services in order to borrow library-owned or licensed assets.

IRI
https://purl.imsglobal.org/caliper/ResourceUseEvent
Term
ResourceUseEvent
Supertype
Event
Properties
The ResourceUseEvent inherits all properties defined by its supertype Event, of which id, type, actor, action, object, and eventTime are required. Profile-specific type restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to the Term ResourceUseEvent. Required
actor Person | IRI The Person who initiated the action. The actor value MUST be expressed either as an object or as a string corresponding to the actor’s IRI. Required
action Term The action or predicate that binds the actor or subject to the object. The value range is limited to Accessed only. Required
object LibraryResource | DigitalResource | IRI The resource that constitutes the object of the interaction. The object value MUST be expressed either as an object or as a string corresponding to the object’s IRI. Required
accessLocation string A non-standardized description of the user's physical location from which the resource was accessed. Examples include: on-campus, off-campus. Optional
