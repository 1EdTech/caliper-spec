ParticipationEvent

A Caliper ParticipationEvent models a Person attending a meeting, lecture, workshop, or any other Library function, whether ad-hoc or scheduled.

IRI
https://purl.imsglobal.org/caliper/ParticipationEvent
Term
ParticipationEvent
Supertype
Event
Properties
The ParticipationEvent inherits all properties defined by its supertype Event, of which id, type, actor, action, object, and eventTime are required. Profile-specific type restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to the Term ParticipationEvent. Required
actor Person | IRI The Person who initiated the action. The actor value MUST be expressed either as an object or as a string corresponding to the actor’s IRI. Required
action Term The action or predicate that binds the actor or subject to the object. The value range is limited to the Attended TODO add other actions? . . . only. Required
object Activity | IRI The Activity that constitutes the object of the interaction. The object value MUST be expressed either as an object or as a string corresponding to the object’s IRI. Required
participationMode string The mode the Person participated in for example virtual, in-person. Optional
forCourse CourseOffering | CourseSection | IRI If attending for a specific CourseOffering or CourseSection it can be referenced here. Optional
arrivalTime DateTime An ISO 8601 date and time value. The value MUST be set to UTC with no offset specified. Optional
