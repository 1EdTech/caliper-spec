NavigationEvent

A Caliper NavigationEvent models an actor traversing a network of digital resources.

IRI
https://purl.imsglobal.org/caliper/NavigationEvent
Term
NavigationEvent
Supertype
Event
Properties
NavigationEvent inherits all properties defined by its supertype Event, of which id, type, actor, action, object, and eventTime are required. Additional profile-specific type and value restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to NavigationEvent. Required
actor Person | IRI The Person who initiated the action. The actor value MUST be expressed either as an object or as a string corresponding to the actor's IRI. Required
action Term The action or predicate that binds the actor or subject to the object. The value range is limited to the actions: NavigatedTo. Required
object DigitalResource | SoftwareApplication | IRI The DigitalResource or SoftwareApplication to which the actor navigated. The object value MUST be expressed either as an object or as a string corresponding to the resource's IRI. Required
referrer DigitalResource | SoftwareApplication | IRI The DigitalResource or SoftwareApplication that constitutes the referring context. The referrer value MUST be expressed either as an object or as a string corresponding to the referrer's IRI. Optional
