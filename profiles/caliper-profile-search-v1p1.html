Events

Search Event

A Caliper SearchEvent models a Person defining a Query and searching a DigitalResource.

IRI
https://purl.imsglobal.org/caliper/SearchEvent
Supertype
Event
Supported actors
Person
Supported actions
Searched
Supported objects
DigitalResource
Properties
SearchEvent inherits all properties defined by its supertype Event. Profile-scoped property type restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to the Term 'SearchEvent'. Required
actor Person The Person who initiated the action. The actor value MUST be expressed either as an object or as a string corresponding to the actor's IRI. Required
action Term The action or predicate that binds the actor or subject to the object. The value range is limited to 'Searched'. Required
object DigitalResource | IRI The DigitalResource searched by the actor. The object value MUST be expressed either as an object or as a string corresponding to the object's IRI. Required
target Query | IRI A Query defined by the actor that constitutes the search criteria. The Query value MUST be expressed either as an object or as a string corresponding to the query’s IRI. Optional

IRI

Supertype
Entity
Properties
Query inherits all properties defined by its supertype Entity. Query is also provisioned with three new properties: author, searched, and searchString. Profile-scoped property type restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to the Term 'Query'. Required
author Person The Person who authored or submitted the Query. The author value MUST be expressed either as an object or as a string corresponding to the author's IRI. Required
searched DigitalResource | IRI The DigitalResource searched by a user with this Query. The searched value MUST be expressed either as an object or as a string corresponding to the resources's IRI. Required
searchString string The search string authored or submitted by the author. Optional
