Properties
SurveyEvent inherits all properties defined by its supertypeEvent. Profile-specific type restrictions are described below:
Property Type Description Disposition
id UUID The emitting application MUST provision the SurveyEvent with a UUID. A version 4 UUID SHOULD be generated. The UUID MUST be expressed as a URN using the form 'urn:uuid:<UUID>' per [[!RFC4122]]. Required
type Term The string value MUST be set to 'SurveyEvent'. Required
actor Person | IRI The Person who initiated the action. The actor value MUST be expressed either as an object or as a string corresponding to the actor's IRI. Required
action Term The action or predicate that binds the actor or subject to the object. The value range is limited to the actions: "WasInvitedToParticipate", "WasRemindedToParticipate", "AgreedToParticipate", or "DeclinedToParticipate". . Required
object Survey | IRI The Survey in which the actor has been invited to participate. The object value MUST be expressed either as an object or as a string corresponding to the resources's IRI. Required

Entities

The SurveyEvent MAY reference a generated SurveyResponse that describes the survey criteria as well as the survey results returned by the query.

SurveyResponse

A Caliper SurveyResponse models a survey engine response to an actor querying a surveyable resource.

IRI
https://purl.imsglobal.org/caliper/SurveyResponse
Supertype
Entity
Properties
SurveyResponse inherits all properties defined by its supertype Entity. SurveyResponse is also provisioned with four additional properties: surveyProvider, surveyTarget, query, surveyResultsItemCount, and surveyResults. Profile-specific type restrictions are described below:
Property Type Description Disposition
id IRI A valid IRI MUST be specified. The IRI MUST be unique and persistent. The IRI SHOULD also be dereferenceable, i.e., capable of returning a representation of the resource. A URI employing the URN scheme MAY be provided in cases where a Linked Data friendly HTTP URI is either unavailable or inappropriate. Required
type Term The string value MUST be set to the Term 'SurveyResponse'. Required
surveyProvider SoftwareApplication | IRI The SoftwareApplication that is the provider of this SurveyResponse. The surveyProvider value MUST be expressed either as an object or as a string corresponding to the resources's IRI. Optional
surveyTarget Entity | IRI The Entity, typically a DigitalResource or SoftwareApplication, that is the target of the survey. The resourceSurveyed value MUST be expressed either as an object or as a string corresponding to the resources's IRI. Optional
query Query | IRI The Query submitted by the actor. Optional
surveyResultsItemCount integer A total count of surveyResults returned. If the Query submitted returned no results the count equal to zero (0). Optional
surveyResults array An ordered collection of entities, typically of type DigitalResource that constitute the results of the survey. Each DigitalResource referenced MUST be expressed either as an object or as a string corresponding to the resource's IRI. Optional

Query

A Caliper Query models survey criteria created by an actor that targets a surveyable resource.

IRI
https://purl.imsglobal.org/caliper/Query
Supertype
Entity
Properties
Query inherits all properties defined by its supertype Entity. Query is also provisioned with three additional properties: creator, surveyTarget, and surveyTerms . Profile-specific type restrictions are described below:
Property Type Description Disposition
id IRI A valid IRI MUST be specified. The IRI MUST be unique and persistent. The IRI SHOULD also be dereferenceable, i.e., capable of returning a representation of the resource. A URI employing the URN scheme MAY be provided in cases where a Linked Data friendly HTTP URI is either unavailable or inappropriate. Required
type Term The string value MUST be set to the Term 'Query'. Required
creator Person | IRI The Person who devised the survey terms comprising this Query. The creator value MUST be expressed either as an object or as a string corresponding to the creator's IRI. Optional
surveyTarget Entity | IRI The Entity, typically a DigitalResource or SoftwareApplication, that is the target of the Query. The resourceSurveyed value MUST be expressed either as an object or as a string corresponding to the resources's IRI. Optional
surveyTerms string The survey terms employed by the creator of this Query. Optional

Representing Caliper as JSON-LD

Individual Caliper events and entity describes are serialized as JSON-LD. IMS Global provides a remote Survey Profile JSON-LD context for mapping SurveyEvent terms to IRIs. Caliper message producers MUST reference this context in every Caliper 1.1 SurveyEvent and/or Query describe created and serialized for transmission to a target endpoint.

See the Caliper 1.1 specification [[!CALIPER-11]] for a discussion of JSON-LD and JSON-LD context handling.
