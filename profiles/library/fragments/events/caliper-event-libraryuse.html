A Caliper LibraryUseEvent models the use of library facilities.

IRI
https://purl.imsglobal.org/caliper/LibraryUseEvent
Term
LibraryUseEvent
Supertype
Event
Properties
The LibraryUseEvent inherits all properties defined by its supertype Event, of which id, type, actor, action, object, and eventTime are required. Profile-specific type restrictions are described below:
Property Type Description Disposition
type Term The string value MUST be set to the Term LibraryUseEvent. Required
actor Person | IRI The Person who initiated the action. The actor value MUST be expressed either as an object or as a string corresponding to the actor’s IRI. Required
action Term The action or predicate that binds the actor or subject to the object. The value range is limited to the Used action only. Required
object Library | LibrarySpace | IRI The Library or LibrarySpace that constitutes the object of the interaction. The object value MUST be expressed either as an object or as a string corresponding to the object’s IRI. Required
presenceDerivedFromSource string A non-standardized description of the source used to indicate presence in a Library. Examples include: wifi, reservation, swipe card, sign in. Optional
presenceDerivedFromID string An identifier that correlates to the presenceDerivedFromSource to indicate which specific source was used if needed, for example, the MAC address of the WiFi router or the ID of the card swipe machine. Optional
