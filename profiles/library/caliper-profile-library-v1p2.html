Abstract

This document extends Caliper 1.2 by defining a Library Profile that describes usage of library facilities and circulation services.

Overview

Design Goals and Rationale

The Caliper Library Profile models a Person, typically a learner, making use of library facilities and circulation services. The profile provides a LibraryUseEvent for recording use of library facilities and a ResourceUseEvent for describing the loaning of library-owned or licensed assets to library patrons.

Caliper provides a controlled vocabulary for enumerating various categories of SystemIdentifier types associated with a Caliper Entity that may prove meaningful when exchanging identifiers between systems (especially with respect to other IMS Global standards).

Serializing the Model

Individual Caliper events and entity describes are serialized as JSON-LD. IMS Global provides a remote Library Profile JSON-LD context for mapping LibraryUseEvent and ResourceUseEvent terms to IRIs. Caliper message producers MUST reference the LibraryUse Profile 1.2-extension context in every Event subtype described in this profile. Likewise, Caliper message producers MUST reference the LibraryUse Profile 1.2-extension context whenever an Entity> subtype described in this profile is emitted on its own as a describe.

See the Caliper 1.2 specification [[!CALIPER-12]] for a discussion of JSON-LD and JSON-LD context handling.
